MX Passdown — Bookmarklet Builder

Encodes the passdown bookmarklet source into a javascript: URL that can be installed as a browser bookmark.

  1. Paste the contents of src/passdown/passdownSource.js into the box below
  2. Click Build Bookmarklet URL
  3. Copy the generated URL from the output box
  4. Paste it into src/passdown/passdownUrl.js as the value of passdownUrl
